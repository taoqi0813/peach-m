# peach-m
基于Vue的移动端ui组件库  

### 安装
```
npm i --save peach-m
```

### 引用
```
import PeachM from "peach-m";
Vue.use(PeachM);
import "peach-m/src/Styles/index.css";
```

### 组件
1. [Switch 开关](./md/SWITCH.md)
2. [Icon 图标](./md/ICON.md)
